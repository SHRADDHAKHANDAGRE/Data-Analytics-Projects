# 📊 Data Analytics Projects Portfolio

## 🧾 Project Objective
The objective of these projects is to analyze real-world datasets, perform data cleaning, exploratory data analysis (EDA), and build interactive dashboards. These projects aim to extract meaningful insights and support data-driven decision-making for business growth.

---

## 📁 Projects Included

### 1. Placement Prediction Dashboard
An end-to-end data analytics and machine learning project to analyze student placement data and predict placement chances.

### 2. Customer Shopping Behavior Analysis
Analysis of customer purchase patterns using SQL and visualization techniques to understand buying behavior.

### 3. Superstore Sales Analysis
Comprehensive sales analysis project focusing on profit, sales trends, and category performance.

### 4. Madhav Store Data Analysis
Retail dataset analysis to identify sales patterns and improve business decisions through dashboards.

---

## 📊 Dataset Used
- Student Placement Dataset  
- Customer Shopping Dataset  
- Superstore Sales Dataset  
- Madhav Store Retail Dataset  

---

## ❓ Sample Questions Answered
- Which factors most influence student placements?  
- What are the key trends in customer purchasing behavior?  
- Which product categories generate the highest sales and profit?  
- Which regions/states contribute the most to revenue?  
- What are the peak sales periods?  
- How can business performance be improved using insights?  

---

## ⚙️ Process

- Cleaned data by handling missing values and inconsistencies  
- Performed Exploratory Data Analysis (EDA) using Python  
- Used SQL for querying and extracting insights  
- Built Machine Learning model (Placement Prediction project)  
- Created interactive dashboards using Power BI  
- Deployed ML model using Streamlit  

---

## 📈 Dashboard

- Interactive Power BI dashboards for all projects  
- Visualized KPIs such as Sales, Profit, Orders, and Trends  
- Used charts like Bar, Line, Pie, and Filters (Slicers)  

---

## 🔍 Project Insights

- Placement chances are highly influenced by CGPA, internships, and skills  
- Customer behavior shows clear patterns in purchasing trends  
- Certain product categories and regions contribute significantly to revenue  
- Sales trends vary across time and channels  
- Data-driven decisions can significantly improve performance  

---

## 🎯 Final Conclusion

These projects demonstrate how data analytics can be used to transform raw data into actionable insights. By combining data cleaning, analysis, visualization, and machine learning, businesses can better understand their customers, optimize performance, and make strategic decisions for growth.

# 📊 Sales & Profit Analysis Dashboard (Superstore Dataset)

## 📌 Overview

This project analyzes a **retail sales dataset** to understand business performance across different regions, categories, and customers.
The dashboard provides insights into **sales, profit, and customer behavior**.

---

## 🎯 Objective

* Track overall sales and profit
* Identify top-performing products and categories
* Analyze regional performance
* Understand customer segments and buying patterns

---

## 🧰 Tech Stack

* **Excel** – Data Cleaning & Preparation
* **Power BI / Excel Dashboard** – Data Visualization

---

## 📊 Dataset

The dataset includes:

* Order ID, Order Date, Ship Date, Ship Mode
* Customer ID, Customer Name, Segment
* Country, City, State, Region
* Product ID, Category, Sub-Category, Product Name
* Sales, Quantity, Profit
* Payment Mode, Returns

👉 [View Dataset](./SuperStore_Sales_Dataset (1).csv)

---

## 🧹 Data Cleaning

* Removed/handled missing values (`#N/A`)
* Formatted date columns
* Standardized categorical values
* Checked duplicates and inconsistencies

---

## 📊 Dashboard Features

* Total Sales, Profit & Quantity KPIs
* Category & Sub-category Analysis
* Region-wise Sales Performance
* Customer Segment Analysis
* Shipping Mode Insights
* Payment Mode Distribution

---

## 🔥 Key Insights

* 📈 **Technology category generates highest profit**
* 🪑 **Furniture has lower profit margins despite good sales**
* 🗂️ **Office Supplies show stable performance**
* 🌎 **West region contributes highest sales**
* 🚚 **Standard Class shipping is most used**
* 💳 **Online & Card payments dominate transactions**

---

## ❓ Business Questions Answered

* Which category is most profitable?
* Which region generates highest sales?
* Which products perform best?
* What shipping mode is most preferred?
* Which customer segment contributes most revenue?

---

## ⚙️ Process

1. Data Cleaning in Excel
2. Data Understanding & Structuring
3. Dashboard Creation (Power BI / Excel)
4. Insight Generation

---

## 💼 Conclusion

This project helps businesses to:

* Focus on **high-profit categories (Technology)**
* Improve **low-margin segments (Furniture)**
* Target **high-performing regions**
* Optimize **shipping and payment strategies**

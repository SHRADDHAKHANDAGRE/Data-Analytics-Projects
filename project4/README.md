# 🛒 Madhav Store Sales Analytics Dashboard

## 📌 Overview

This project analyzes **Madhav Store’s online sales data across India** to generate meaningful insights and support business decision-making.
The analysis is performed using **data cleaning and visualization techniques**.

---

## 🎯 Objective

* Track overall sales & profit performance
* Identify top-performing categories and products
* Analyze customer and regional trends
* Detect loss-making areas

---

## 🧰 Tech Stack

* **Python (Pandas)** – Data Cleaning & Preparation
* **Power BI** – Dashboard & Visualization

---

## 📊 Dataset

The project uses two datasets:

### 1. Orders Data

* Order ID, Order Date, Customer Name
* State, City

👉 [View Orders Dataset](./orders.csv)

### 2. Sales Data

* Order ID, Amount, Profit, Quantity
* Category, Sub-Category, Payment Mode

👉 [View Sales Dataset](./sales.csv)

---

## 🔗 Data Preparation

* Cleaned and formatted datasets using Python
* Handled missing values and inconsistencies
* Merged both datasets using **Order ID**
* Loaded final dataset into Power BI for analysis

---

## 📊 Dashboard Features

* Total Sales, Profit & Quantity Overview
* Category & Sub-category Performance
* State & City-wise Sales Analysis
* Payment Mode Distribution
* Profit vs Loss Analysis
* Monthly Trends

---

## 🔥 Key Insights

* 📈 **Electronics category generates highest profit**
* 🪑 **Furniture has high sales but lower profit margins**
* 👗 **Some clothing products show negative profit**
* 🏙️ **Maharashtra (Pune & Mumbai) leads in sales**
* 💳 **Credit Card & EMI are most preferred payment modes**
* 📉 Presence of loss-making transactions indicates scope for optimization

---

## ❓ Business Questions Answered

* Which category is most profitable?
* Which regions generate highest sales?
* Which products are causing losses?
* What payment methods do customers prefer?
* How does sales vary across locations?

---

## ⚙️ Process Followed

1. Data Cleaning using Python
2. Data Merging (Orders + Sales)
3. Data Visualization in Power BI
4. Insight Generation

---

## 💼 Conclusion

This dashboard helps Madhav Store to:

* Focus on **high-profit categories**
* Reduce **loss-making products**
* Target **high-performing regions**
* Improve **business strategy using data insights**
